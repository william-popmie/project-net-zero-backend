## Lokale setup (Python)

Dit project gebruikt een virtual environment in `.venv`.

### 1) Activeer de virtual environment (PowerShell)

```powershell
.\.venv\Scripts\Activate.ps1
```

### 2) Controleer Python

```powershell
python --version
```

### 3) Gebruik environment variables

Kopieer het voorbeeldbestand:

```powershell
Copy-Item .env.example .env
```

Vul daarna je echte variabelen in `.env` in.
